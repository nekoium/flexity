What Flexity is

A structure that doesn't fight the task.

Knowledge work keeps reshaping itself — today's project plan is tomorrow's checklist, then next week's SOP. Flexity holds one recursively composable object model underneath all of them, so the view follows the work instead of the other way around.

The model

Four elements. Clear stacking rules.

Everything is JSON. Everything composes. The rules below are the whole grammar — what each element can host.

Board root
  • Hosts columns & trays
  • One board = one file
  • The root container
Column
  • Hosts trays & notes
  • Takes a full vertical slot
  • Cannot be nested — board-level only
Tray
  • Hosts trays & notes
  • Recursively composable
  • Flexible height — a pinned tray is a column
Note
  • A markdown slip with full markdown
  • Stacks with spacing, like list levels
  • Cannot host other notes

Features

Beyond a plain Kanban.

Standard boards stop at columns and notes. Flexity adds the mechanics knowledge work actually needs.

Stackable notes

Notes stack under notes to any depth. Break a task into sub-notes without leaving the board or duplicating data.

Shadows

Cast a note as a read-only pointer into another column — keep a task in its project while it shows up in today's work. Hover either to highlight the pair.

Note references

Link notes by title inline, or pick one from a list. References stay live — move the host and the link follows.

Foldable items

Collapse any column or board to its title. Keep the structure visible without the noise when you need the overview.

Switch column ⇄ tray

A column is basically a tray that can't be nested. Promote a root tray to a column to focus on it, demote it back when the moment passes.

Highlight to focus

Mark the notes that matter today. One keystroke toggles a calm highlight so the important surfaces above the rest.

Markdown notes

Each note is markdown by itself — or a mirror of another note in your vault. The body is prose, not a label.

Undo, always

Every mutation goes through history. Ctrl+Z walks it back. Experiment without consequence.
